# com103-midterm-exam
Overview of the Task Tracker Program:

This Python program is designed to manage and track tasks for a group project in a COM 103 midterm exam. It allows users to:

Input Project Details: Users provide the project title and group name.
Display Task Information: The program lists five predefined tasks with their corresponding hours.
Assign Tasks: Users assign tasks to group members, set their status (either "done" or "pending"), and automatically calculate points based on the task's status (2 points for "done" and 1 for "pending").
Track Progress: It calculates the total points earned and the progress percentage based on completed tasks.
Display Task Board: The program generates a detailed report showing each task's name, assigned member, status, points, and total project progress.
