## Overview

The **COM 103 Task Tracker** is a Python program designed to help manage and track tasks for a group project in a midterm exam setting. It allows users to:

- **Assign tasks** to group members.
- **Track task completion** based on a "done" or "pending" status.
- **Calculate total points** earned and overall progress as a percentage.
- **Display detailed task information** for easy project monitoring.

### Features:
- **Task Management**: Lists predefined tasks and their associated hours.
- **Progress Tracking**: Automatically calculates progress based on task completion and displays overall project status.
- **Badges**: Recognize and reward progress with customizable badges.

---

## Badges

The program assigns various badges based on the project’s overall completion and task status. These badges provide visual indicators of project progress and individual task completion.

### Project Progress Badges:
- **"On Fire!"**: Awarded when the project is 100% complete.
- **"Steady Worker"**: Awarded when the project is 75%-99% complete.
- **"Needs Improvement"**: Awarded when the project is below 50% complete.

### Task Completion Badges:
- **"Task Master"**: Awarded when **all** tasks are completed ("done").
- **"Halfway There"**: Awarded when **half or more** tasks are marked as "done".
- **"Getting Started"**: Awarded when fewer than half of the tasks are completed.
